接口说明

所有教室的所有课程

api地址
http://1.14.180.202:8080/v1/course/list
参数说明
名称方法类型参数
课程列表get无

获取某教室一天的课程

api地址
http://1.14.180.202:8080/v1/course/course_cw?room_id=x&day=y
参数说明
名称方法类型参数
教室日课程getroom_id：教室id ; day：日时间

示例：

获取教室id为0并且周1的课程

http://1.14.180.202:8080/v1/course/course_cw?room_id=0&day=1

获取教室一周的课程

api地址
http://1.14.180.202:8080/v1/course/course_id?room_id=x
参数说明
名称方法类型参数
教室周课程getroom_id：教室id

示例：

查询教室id=4一周的课程

http://1.14.180.202:8080/v1/course/course_id?room_id=4

添加课程

api地址
http://1.14.180.202:8080/v1/course/new
参数说明
名称方法类型参数
添加课表postname：课程名称
is_bi_week：是否单双周
course_id：课程id
teacher：教师
student_total：上课人数
time：上课时间(例如：第1-2节)
class_room：教室名称(例如：D7尚美楼308)
class_room_id：教室id
week：周时间(例如：3)
week_nums：课程时长(例如：第1-16)
