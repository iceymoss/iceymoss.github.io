# 技术分析基础

技术分析的核心假设是：市场价格已经反映了所有相关信息，且历史模式会重复出现。常见的指标包括移动平均线（Moving Averages）、相对强度指数（RSI）、移动平均收敛散度（MACD）等。这些指标可以帮助我们识别趋势、超买超卖状态和市场动量。

对于开发者来说，技术分析不仅仅是理论，更可以通过编程实现自动化分析。在接下来的部分，我们将使用Python，因为它易于上手且拥有丰富的库（如pandas、numpy、matplotlib）。让我们从一个简单的例子开始：获取以太坊的历史价格数据。

## 计算简单移动平均线（SMA）

移动平均线是技术分析中最基本的指标之一，它平滑价格数据以识别趋势。我们计算两个SMA：短期（例如10天）和长期（例如30天）。

```python
# 计算10天和30天的简单移动平均线
df['SMA_10'] = df['Close'].rolling(window=10).mean()
df['SMA_30'] = df['Close'].rolling(window=30).mean()

# 绘制价格和移动平均线
plt.figure(figsize=(12,6))
plt.plot(df['Close'], label='以太坊收盘价', color='blue', alpha=0.7)
plt.plot(df['SMA_10'], label='10天SMA', color='orange', linestyle='--')
plt.plot(df['SMA_30'], label='30天SMA', color='green', linestyle='--')
plt.title('以太坊价格与移动平均线')
plt.xlabel('日期')
plt.ylabel('价格（美元）')
plt.legend()
plt.grid(True)
plt.show()
```

当短期SMA上穿长期SMA时，可能表示上涨趋势开始；下穿时，可能表示下跌趋势。但请注意，这只是一个简单信号，需要结合其他指标验证。

## 计算相对强度指数（RSI）

RSI用于衡量价格的超买或超卖状态，范围在0到100之间。通常，RSI超过70表示超买，低于30表示超卖。

```python
def calculate_rsi(prices, window=14):
    """计算RSI指标"""
    delta = prices.diff()  # 计算价格变化
    gain = (delta.where(delta > 0, 0)).rolling(window=window).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(window=window).mean()
    rs = gain / loss
    rsi = 100 - (100 / (1 + rs))
    return rsi

df['RSI'] = calculate_rsi(df['Close'])

# 绘制RSI
plt.figure(figsize=(12,6))
plt.plot(df['RSI'], label='RSI', color='purple')
plt.axhline(y=70, color='red', linestyle='--', label='超买线 (70)')
plt.axhline(y=30, color='green', linestyle='--', label='超卖线 (30)')
plt.title('以太坊RSI指标')
plt.xlabel('日期')
plt.ylabel('RSI值')
plt.legend()
plt.grid(True)
plt.show()
```

RSI可以帮助我们识别潜在的反转点，但像所有指标一样，它并非万能，应谨慎使用。

## 进阶：结合多个指标进行简单预测

我们可以将SMA和RSI结合起来，生成一个简单的交易信号。例如，当SMA_10上穿SMA_30且RSI低于70时，考虑买入信号。

```python
# 生成信号
df['Signal'] = 0  # 0表示无信号，1表示买入，-1表示卖出
df.loc[(df['SMA_10'] > df['SMA_30']) & (df['RSI'] < 70), 'Signal'] = 1  # 买入信号
df.loc[(df['SMA_10'] < df['SMA_30']) & (df['RSI'] > 30), 'Signal'] = -1  # 卖出信号

# 可视化信号
plt.figure(figsize=(12,6))
plt.plot(df['Close'], label='以太坊收盘价', color='blue', alpha=0.7)
plt.plot(df['SMA_10'], label='10天SMA', color='orange', linestyle='--')
plt.plot(df['SMA_30'], label='30天SMA', color='green', linestyle='--')
# 标记买入和卖出点
buy_signals = df[df['Signal'] == 1]
sell_signals = df[df['Signal'] == -1]
plt.scatter(buy_signals.index, buy_signals['Close'], color='green', marker='^', label='买入信号', s=100)
plt.scatter(sell_signals.index, sell_signals['Close'], color='red', marker='v', label='卖出信号', s=100)
plt.title('以太坊技术分析信号')
plt.xlabel('日期')
plt.ylabel('价格（美元）')
plt.legend()
plt.grid(True)
plt.show()
```

这个策略非常基础，实际应用中需要回测和优化。作为开发者，我们可以进一步使用机器学习方法，如LSTM（长短期记忆网络），来进行更复杂的预测。

## 进阶：使用LSTM进行价格预测

LSTM是一种递归神经网络，适用于时间序列预测。这里提供一个简化示例，使用历史价格预测未来价格。

```python
import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense
from sklearn.preprocessing import MinMaxScaler

# 数据预处理
scaler = MinMaxScaler(feature_range=(0,1))
scaled_data = scaler.fit_transform(df['Close'].values.reshape(-1,1))

# 创建训练数据：使用过去60天的价格预测下一天
train_size = int(len(scaled_data) * 0.8)
train_data = scaled_data[:train_size]
test_data = scaled_data[train_size:]

def create_dataset(data, time_step=60):
    X, y = [], []
    for i in range(len(data)-time_step-1):
        X.append(data[i:(i+time_step), 0])
        y.append(data[i+time_step, 0])
    return np.array(X), np.array(y)

time_step = 60
X_train, y_train = create_dataset(train_data, time_step)
X_test, y_test = create_dataset(test_data, time_step)

# 重塑数据为LSTM输入格式 [样本数, 时间步长, 特征数]
X_train = X_train.reshape(X_train.shape[0], X_train.shape[1], 1)
X_test = X_test.reshape(X_test.shape[0], X_test.shape[1], 1)

# 构建LSTM模型
model = Sequential()
model.add(LSTM(50, return_sequences=True, input_shape=(time_step, 1)))
model.add(LSTM(50, return_sequences=False))
model.add(Dense(25))
model.add(Dense(1))
model.compile(optimizer='adam', loss='mean_squared_error')

# 训练模型（这里为了演示，使用少量轮次）
model.fit(X_train, y_train, batch_size=32, epochs=10, verbose=1)

# 预测
predictions = model.predict(X_test)
predictions = scaler.inverse_transform(predictions)  # 反归一化

# 可视化预测结果
actual_prices = scaler.inverse_transform(y_test.reshape(-1,1))
plt.figure(figsize=(12,6))
plt.plot(actual_prices, label='实际价格', color='blue')
plt.plot(predictions, label='预测价格', color='red', linestyle='--')
plt.title('以太坊价格LSTM预测')
plt.xlabel('测试集时间步')
plt.ylabel('价格（美元）')
plt.legend()
plt.grid(True)
plt.show()
```

LSTM模型需要大量数据和调优，这里仅为入门示例。在实际项目中，你还需要考虑特征工程、模型验证和风险管理。

# 注意事项

技术分析并非完美，它基于历史数据，无法预测突发事件（如监管变化或黑客攻击）。以太坊市场尤其复杂，因为其生态不断演进。作为开发者，我们应该：
- 结合基本面分析，例如关注以太坊网络升级（如EIP-1559）。
- 使用多个指标交叉验证，避免依赖单一信号。
- 实践回测，在历史数据上测试策略，但记住过去表现不代表未来结果。
- 始终保持谦虚，承认预测的局限性，并注重风险管理。
