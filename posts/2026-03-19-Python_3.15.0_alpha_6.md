Python 语言正处于 3.15 版本的早期开发阶段，当前发布的 3.15.0 alpha 6 是八个计划中的第六个 alpha 版本。Alpha 阶段旨在方便测试新功能、修复漏洞以及验证发布流程，但此阶段功能在 Beta 阶段开始前（2026-05-05）仍可能增减，在候选发布阶段前（2026-07-28）仍可能修改或删除，因此不推荐用于生产环境。

与 3.14 系列相比，3.15 系列规划并已实现多项重大新特性。核心技术创新包括：1) **PEP 799**：引入了一个全新的高频、低开销的统计采样分析器及专用的性能分析包，旨在提供更高效的性能剖析能力。2) **PEP 798**：在推导式中支持使用 `*` 和 `**` 进行解包操作，这将增强推导式的表达能力和代码简洁性。3) **PEP 686**：将 UTF-8 设置为 Python 的默认文本编码，统一了跨平台行为，解决了长期以来因默认编码不一致导致的兼容性问题。4) **PEP 782**：新增了 `PyBytesWriter` C API，用于更高效地创建 Python 字节对象。5) **PEP 728**：扩展了 `TypedDict` 的功能，支持定义带类型注解的额外项。6) **JIT 编译器重大升级**：对即时编译器进行了显著优化，在 x86-64 Linux 平台上相比标准解释器实现了 3-4% 的几何平均性能提升，在 AArch64 macOS 平台上相比尾调用解释器实现了 7-8% 的速度提升。7) **改进的错误信息**：提升了错误信息的可读性和帮助性。

这些改进标志着 Python 在性能剖析工具、语法便利性、国际化支持、底层 API 以及运行时性能等多个关键领域的持续演进。采样分析器和 JIT 的升级直接针对高性能计算和大型应用场景，而 UTF-8 默认编码和推导式解包则提升了开发体验和代码可维护性。下一个预发布版本 3.15.0a7 计划于 2026-03-10 发布。
